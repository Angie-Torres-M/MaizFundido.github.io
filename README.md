# MazFundido
Maíz Fundido – Sitio Web de Impresión 3D
Maíz Fundido es un proyecto de desarrollo web que presenta los productos de una marca mexicana de impresión 3D con un enfoque artesanal, colorido y sustentable.
El sitio combina diseño visual atractivo, estructura responsive y componentes reutilizables para ofrecer una experiencia dinámica y amigable para el usuario.

Tecnologías utilizadas
HTML5 – estructura semántica del sitio
CSS3 – personalización de estilos y animaciones
Bootstrap 5 – diseño responsive y componentes visuales
JavaScript (ES6) – interactividad y control del DOM

En desarrollo / Próximos pasos
Integración con backend en Java (Spring Boot)
Conexión con base de datos MySQL para la gestión de productos y usuarios
Implementación de API REST para el catálogo y cotizaciones

Identidad visual
El diseño está inspirado en la estética alegre y artesanal del emprendimiento, con una paleta de colores cálidos y vibrantes (amarillo maíz, verde hoja, rosa alegre y azul cielo), reflejando la esencia creativa y sustentable de la marca.

Objetivo
Construir una plataforma digital que comunique la personalidad de Maíz Fundido, facilite el contacto con clientes y siente las bases para una futura tienda en línea.
